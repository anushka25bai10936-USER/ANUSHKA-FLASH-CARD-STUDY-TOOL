
PROJECT DESCRIPTION:: 
Often, going through a chapter while studying feels too boring and a tiresome task.To make it engaging and productive flash card note taking with 5 step study method is actually a good solution. The Flashcard Study Tool is an interactive is a study assistant designed to help students learn more effectively. It combines self made or customised flashcards, self-testing, automated accuracy tracking, and a 5-step study method : Survey,Question,Read,Recite and Review The program allows users to input their own study material, test themselves, redo incorrect answers, and visualize their performance using graphs or histograms . It also includes randomized flashcard ordering and accuracy statistics using Pythonâ€™s built-in modules.

FEATURES:: 
FlashCard MODE :

Create your own set of flashcards Randomized question order using the RANDOM MODULE . Self-evaluation: Mark each answer as CORRECT OR INCORRECT.

CHOICE to retry or Redo the questions you got INCORRECT.

Automatic scoring and feedback.

VISUAL BAR GRAPH of correct and incorrect answers using MATPLOTLIB MODULE.

Calculates Average Accuracy using the STATISTICS MODULE.

READING AND REVISION MODE::

A 5-step process:: SURVEY : Go through the chapter and write down your own notes. QUESTION : list questions about the topic. READ : read while keeping your questions in mind. RECITE : summarize what you've learned. REVIEW : revisit your notes and answers.

GRAPH VISUALISATION::

Displays a BAR CHART showing: Number of CORRECT ANSWERS Number of INCORRECT ANSWERS

USER FRIENDLY INTERACTION::

Simple and sorted interface. Clear instructions and guidance for each step. Infinite practice loops until the user chooses to stop.
